HTML Table Elements

Element Name

Element Tag

Element Definition
Table <table> Used to create tables. The element that contains all HTML tables.
Table Head <thead> Rows used to create, section off and contain all table headings.
Table Body <tbody> Rows used to create, section off and contain all content in the table.
Table Row <tr> Used to create and contain rows in a table.
Table Data <td> Used to enter data into table rows.
Table Footer <tfoot> Rows that create footers at the end of the table. They do not form part of the table
